
# 📖 EnglishAdventures 

Welcome to the official repository for **EnglishAdventures**, a premium landing page designed to inspire students to explore the English language through literature and personalized coaching.

## ✨ About the Project
EnglishAdventures is built for learners who want to move beyond basic grammar. This site highlights our unique "Literary Free Talk" program and offers clear pricing for our community club and private elite coaching.

### Key Features:
* **Modern "Vava-Voom" Design:** A clean, responsive layout with high-definition boundaries.
* **Literary Free Talk Section:** Showcasing our storytelling-based approach to fluency.
* **Interactive Pricing:** Clearly defined tiers for Group Club Members ($90) and Private Coaching ($150).
* **Direct Contact:** Integrated "Email Me" functionality for instant student inquiries.

## 🛠️ Built With
* **HTML5** - For the structure and content.
* **CSS3** - For the custom "Sky Blue" styling, glassmorphism headers, and textured backgrounds.
* **Google Fonts** - For professional typography.
